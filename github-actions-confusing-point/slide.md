## GitHub Actions とは

GitHub 公式の CI/CD の仕組み。  

コードの変更やプッシュが行われたときに、特定のワークフローを自動的に実行できる。  

---

## 使い方


---

.github/workflows ディレクトリに、yamlファイルを配置。  

![x](assets/001.png)

---

こんな感じでジョブが動く。  

![x](assets/002.png)

---

使うだけなら超簡単。  

しかし、やりたい事を実現させるには、結構変なハマり所もある。
